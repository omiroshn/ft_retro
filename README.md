# ft_retro

### Description

*The goal of this project is to implement a simplistic shoot-em-up-style game in your terminal.*

### Requirements
- linux or macos
- installed ncurses

How to start the program:
```
make
./ft_retro
```

Some usefull buttons:
```
q - exit
Arrows - moving
Space - shoot
```
